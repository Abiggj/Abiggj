## Projects

### CraftTrade
CraftTrade is a simulation tool that uses adversarial neural networks, natural language processing, and LSTM to analyze financial OHLCV data of stocks and main news headlines. This project demonstrates my ability to integrate AI and time series analysis to create sophisticated financial models.

### TrulyPresent
TrulyPresent is a mobile application that leverages facial recognition technology to automate classroom attendance. This project showcases my expertise in computer vision and AI, aiming to streamline the attendance process and enhance classroom management.

### Evolve Entertainment
Evolve Entertainment is an app designed to revolutionize event management. It includes features like Home, Search, Categories, My Events, Create Event, Notifications, Profile, Event Details, and Ticketing, providing a seamless user experience for event discovery and management.

### Re-dot
Re-dot is a unique application that blends the features of Reddit and Tinder to enhance user engagement. It aims to create an interactive and engaging platform for users to share and discover content in an innovative way.

### Sudoku Solver
A Python-based package for solving Sudoku puzzles, featuring a `solverr` class with methods for checking rows, columns, boxes, and possibilities to efficiently solve puzzles.
